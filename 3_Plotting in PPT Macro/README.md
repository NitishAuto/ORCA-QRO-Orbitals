# Step 3: PowerPoint Macro

This is a VBA macro that takes the images and data from Chimera and arranges them neatly into PowerPoint slides.

### Setup & Usage:
1. Open PowerPoint, press `ALT + F11` to open the VBA editor.
2. Go to `File -> Import File` and select `MOSlideGenerator.bas`.
3. Close the editor, press `ALT + F8`, and run `Make_MO_Slide_FINAL`.
4. It'll pop up a folder picker. Just point it to the folder containing your `.png` images and `gopal.txt`.
5. The macro will automatically generate slides with a 4-column layout for all your MOs.
